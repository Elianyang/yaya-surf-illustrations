# 示例说明

这里原本放过一张手写 SVG 结构草图，用来临时说明“怕错过直播潮”的构图逻辑。

但那张图只是占位草图，人物比例和完成度不适合放在 GitHub 首页当正式示例，所以已经从 README 主视觉里撤下。

正式示例建议放真实生成后的 PNG：

- `01-fomo-livestream.png`：我不是没时间学习，我是怕错过。
- `02-ai-digital-labor.png`：AI 终于开始替我干数字苦力了。

正式示例图需要遵守两个角色细节：

- 丫丫是一个马尾辫/单条辫子，不是双马尾或双辫。
- 底角要有小字署名：`Co-Created with 丫丫老师AI研习社`。

如果当前环境里的图片生成结果只显示在对话中、没有落盘，可以在 Codex/ChatGPT 里下载满意的图片后放到这个目录，再在 README 中替换图片链接。
